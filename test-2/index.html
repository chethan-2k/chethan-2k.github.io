If you’re self hosting Ghost, you’ll need to configure a few things. If you’re using Ghost(Pro) you can ignore this step (and you probably want to check out the Getting Started with Ghost(Pro) Guide rather than this one.

Configuring your blog takes place in the config.js file. This file is created the first time you start Ghost and lives in the root directory of Ghost. If you followed the linux install guide, this will be located at /var/www/ghost/config.js. If you haven’t run Ghost yet, you can run the command cp config.example.js config.js to create a config.js file.
